# Swifty Companion

A Flutter mobile app that lets you search for 42 Network students and view their profile information using the 42 API with OAuth2 authentication.

## Features
- Search any 42 student by login
- View profile details (login, email, level, wallet, location...)
- View skills with level and percentage
- View completed and failed projects

## Setup
1. Clone this repo
2. Create a `.env` file in the root with:
- UID=your_client_id
- SECRET=your_client_secret
3. Run `flutter pub get`
4. Run `flutter run`

*(env example can be found on .env-example file)* 

## Tech Stack
- Flutter (Dart)
- 42 API (OAuth2)
